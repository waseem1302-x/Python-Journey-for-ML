### 2. Data Types

* `int`, `float`, `str`, `bool`
* Use `type()` to check the type.

### 3. Input and Output

* `input()` is used to take input from the user.
* `print()` displays output.


* Convert one type to another using `int()`, `float()`, `str()`.
